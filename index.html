fileasy

Client‑side file utilities: convert, inspect, download, and more.

npm • Docs • GitHub

Key features

  • Convert images: svgToPng, jpegToPng, etc.
  • File & buffer conversion: fileToBase64, base64ToFile, binaryToString
  • Get file metadata: getInfoFromFile, getMimeTypeFromBuffer
  • Download and select files in the browser

Quick Start

npm install fileasy

Then:

import { fileToBase64, base64ToFile } from 'fileasy';

const base64 = await fileToBase64(myFile);
const restored = base64ToFile(base64, 'restored.png', 'image/png');

See more usage examples on the Usage and API pages.
